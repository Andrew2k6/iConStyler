About iConStyler.
  • Development

    iConStyler has been developed by Eleve11 (read eleven) Creative Commons Licence.

    It all started when the jailbreak scene seemed to be in crisis that I started theming my device for myself. Then on Sat Jan 19 I thought it would have been awesome if everyone could install a themed icon for all the apps they want. So I started programming this web page... Have a look around it.
